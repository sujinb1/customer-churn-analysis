# Customer Churn Analysis

## Project Overview
visualize the data and answer the following questions

## Questions
-Does contract type affect customer churn?
-How do monthly charges impact customer retention?
-Does customer tenure influence churn behavior? 

## Technologies Used
- Python
- Pandas
- Matplotlib

## Steps Performed
. Import libraries
. Data Cleaning
. Exploratory Data Analysis

## Buisness insight
-Customers with month-to-month contracts churn more.
-Higher monthly charges increase churn probability.
-Long tenure customers are less likely to churn.

## Result
data analysis and found solutions for the questions
